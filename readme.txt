== Installation ==
1.  Install WooCommerce plugin and configure your store (if you haven't done so already - http://wordpress.org/plugins/woocommerce/).
2.  Install "Wechat Crossborder for WooCommerce" wordpress plugin just like any other Wordpress plugin.
3.  Activate.
4. Setup your weChat Crossborder merchant id, key, throuth <strong>Woocommerce -> Settings -> Payment Gateways -> WeChatPay</strong>.
   You can apply wechat payment through https://pay.weixin.qq.com/index.php/public/wechatpay/home
5. Currency set for the currency of your WeChat merchant.
